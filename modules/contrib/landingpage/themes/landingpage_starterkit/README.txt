To set your custom theme for Landing Page Framework:

- copy this folder in /theme in your Drupal root folder
- rename the folder 'landingpage_starterkit', for example, to 'my_theme'
- rename THEMENAME.starterkit.yml to my_theme.info.yml
- rename THEMENAME.libraries.yml to my_theme.libraries.yml
- rename THEMENAME.libraries.yml to my_theme.libraries.yml

Go Ahead!

- customise css, js and templates 
- turn on theme in admin
- set this theme for your Landingpage

You can add set of your classes in the .info.yml file of your theme and they will be supported in LandingPage Classes autocomplete. See on "landingpage classes:" settings.
